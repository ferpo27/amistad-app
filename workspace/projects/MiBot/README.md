# MiBot
## Introducción
Este proyecto es un ejemplo de cómo crear un bot utilizando tecnologías de inteligencia artificial.

## Requisitos previos
* Tener instalado Python 3.8 o superior
* Tener instalado el entorno virtual de Python (opcional)

## Instalación
1. Clonar el repositorio utilizando `git clone https://github.com/usuario/repo.git`
2. Instalar las dependencias utilizando `pip install -r requirements.txt`

## Ejecución
1. Activar el entorno virtual de Python (si se utiliza)
2. Ejecutar el comando `python main.py` para iniciar el bot

## Configuración
* Editar el archivo `config.json` para configurar las opciones del bot
